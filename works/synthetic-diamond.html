Project file / symbolic material study

03

Synthetic Diamond

A pharmaceutical tablet is reframed as a synthetic gem, questioning value, hardness, desire, and symbolic legitimacy.

Year
2024
Medium
Metal, tablet-like form, belt buckle, ring
Material
Metal structure, pharmaceutical-tablet reference, geometric setting
Type
Jewelry / object / symbolic material study
Size
Variable

01

Overview

Synthetic Diamond treats a tablet-like form as a substitute gem. The work connects pharmaceutical symbolism with the visual language of jewelry, asking how value is produced through setting, display, hardness, and belief. Instead of presenting the object as decoration, the project examines how systems of value become wearable and socially legible.

02

Tablet as gem

A pharmaceutical reference is placed inside the logic of jewelry. The tablet-like form becomes a substitute stone rather than a medical object.

03

Belt buckle

The belt buckle shifts the object toward the body and public display. Value is carried through a functional object associated with fastening and presentation.

04

Ring

The ring format concentrates the tablet reference into a familiar jewelry setting. The work keeps the object between ornament, sign, and material question.

05

Symbolic hardness

The project treats hardness as a cultural claim rather than only a physical property. Belief, display, and setting produce the object as valuable.
