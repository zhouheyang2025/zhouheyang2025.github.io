- Use the Image ID as the stable reference for editing.
- Do not reference raw image files directly from `source-indesign/01-raw-package/Links/` in website pages.
- Copy selected images into `assets/images/`, grouped by project.
- Use lowercase English filenames with hyphens when final website filenames are chosen.
- Update this manifest before changing image order, captions, layout classes, or website filenames.
- Each project `<figure>` should include the matching `data-image-id` attribute.
