# Layout Notes

The PDF portfolio is horizontal and spread-based. The website should translate the PDF's visual hierarchy into responsive web layouts.

Do not copy the PDF layout literally.

Preserve:
- image hierarchy
- image grouping
- dominant / secondary image relationships
- process sequence
- object-detail rhythm
- restrained archival tone

Adapt:
- fixed A4 spread composition
- overly dense print layout
- images that are too small for screen
- text blocks that need to become readable on web

Desktop / landscape:
- Keep a horizontal, spread-like feeling.
- Use wide image bands, two-column image/text relations, and controlled grids.
- Align with the Fish and Rice page structure.

Mobile:
- Stack images vertically.
- Keep captions close to images.
- Avoid tiny multi-column images.
- Preserve reading order over exact PDF layout.
