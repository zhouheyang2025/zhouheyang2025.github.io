# DESIGN_BRIEF.md

## Website concept

This website is a digital archive of artistic practice.

It should present works as spatial, material, and conceptual investigations rather than as commercial projects.

The visual system should be calm, cold, and controlled.

## Layout direction

The site should use:

- strong grid
- clear margins
- small metadata
- large image fields
- restrained typography
- project index structure
- page layouts that feel like catalogues or archival sheets

## Reference translation

A reference like tiagocanario.com can be used only as a loose reference for a personal digital archive feeling.

Do not copy its layout directly.

Translate the reference into a colder, more forensic, more art-school-oriented artist portfolio.

## Homepage direction

The homepage should contain:

- artist name
- short one-sentence practice statement
- project index
- navigation to Works / About / Contact
- one or two strong visual entries
- no decorative hero slogan

## Works page direction

The Works page should feel like an archive index.

Each project should show:

- project number
- title
- year
- medium
- short description
- thumbnail or documentation image

## Project page direction

Each project page should contain:

- title
- one-sentence concept
- year / medium / material / size / type
- short project text
- image sequence
- process documentation
- details
- optional video or 3D model section

## Writing style

Text should be:

- concise
- calm
- precise
- analytical
- not poetic in an excessive way
- not commercial
- not over-explanatory

The tone should be closer to project documentation than self-promotion.
